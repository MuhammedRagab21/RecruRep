# RecruRep Landing Page — Constitution

## Mission
Build a landing page for RecruRep, a SaaS that lets non-native English teachers apply to teaching jobs with one click — eliminating the need to bounce between apps, websites, and Facebook groups.

## Design Principles
- **Dark-first**: Single dark canvas (#101010) with emerald green (#00d992) accent. No light mode.
- **Precision over decoration**: Hairline borders, no shadows, restrained typography.
- **Teacher-first messaging**: Empathetic, clear, benefits-driven copy. Non-native English teachers are the audience — write for them.
- **Mobile-first responsive**: Teachers browse on phones. Every section must work at 320px.

## Technical Principles
- **Vanilla stack**: Pure HTML/CSS/JS. No frameworks, no build step.
- **Self-contained**: Single index.html with embedded CSS + JS for easy deployment.
- **Performance**: Under 100KB total, no external dependencies, fast first paint.
- **Accessibility**: Semantic HTML, proper contrast ratios, keyboard-navigable.

## Content Principles
- Value proposition clear in first 3 seconds.
- Every section answers "what's in it for the teacher?"
- Social proof from real teacher scenarios.
- Pricing is transparent and upfront.
